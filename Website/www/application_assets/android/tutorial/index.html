Welcome to Thoth

Thoth is Optical Character Recognition (OCR) Application for android that can processed photographed images of typewritten or printed text into machine-encoded/computer-readable text. There are a few basics steps to using Thoth

Tutorial

Directions

  1. Press the capture button the next screen
  2. Capture an image and wait for processing
    • Project Thoth is still in beta, so please make sure the images are taken in a lit room with a clear/focused view of the image.
    • The current version of Thoth is intended for clear printed english text. It may struggle with numbers and symbols. We are working hard on introducing a wide array of fonts.
  3. Copy the OCR text and use to your liking.
